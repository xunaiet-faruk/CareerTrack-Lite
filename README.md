# 🚀 CareerTrack Lite – Smart Job Application Tracker



> **CareerTrack Lite** is a modern MERN Stack web application that helps job seekers organize, monitor, and manage their job applications efficiently. It provides secure authentication, application tracking, analytics, and an intuitive dashboard to simplify the job search process.

# 📌 Why MongoDB?

Although the project recommendation mentioned PostgreSQL with Prisma, I selected **MongoDB** because of my practical experience with the MERN Stack. This allowed me to build the project faster while maintaining secure authentication, authorization, CRUD operations, and a smooth user experience.
